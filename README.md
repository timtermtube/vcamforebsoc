# Outdated
본 프로젝트는 현재(21-08-24) 기준으로 사용이 어렵거나 불가합니다.

# 설명
다른 기기의 카메라를 연결시켜 사용하고자 하는 컴퓨터의 브라우저에서 사용하게 해줌

# 필요한 것
- HTTPS 인증서
- 카메라를 지원하는 디바이스와 컴퓨터 1대

# 안내사항 
- 코드 내 전역 변수 exceptions에 로딩이 간혹 되지 않는 일부 사이트를 추가하시거나 사용을 원치 않으시는 사이트를 추가하실 수 있습니다.
- 마이크와 같은 음성 입력 장치를 지원하지 않습니다. 원하신다면 따로 마이크를 설치하셔야 합니다.
- 일부 사이트에서 CORS(Cross-origin resource sharing) 문제로 작동하지 않는 경우가 있습니다.


# 사용 방법
npm i express peer (라이브러리가 미설치될 시에만)
sudo node index.js
